# Mission: NotebookLM MCP Server Setup

대표님의 소중한 NotebookLM 데이터를 연동하기 위해, 코다리 부장이 직접 4단계 작전을 지휘합니다.

**[작업 목표]**
NotebookLM MCP 서버 설치, 설정, 인증 및 최종 연동 확인.

**[수행 단계]**

**1. 설치 (Installation)**
* **지시:** 공식 GitHub 저장소(**`https://github.com/wonseokjung/notebooklm-mcp`**)를 참조하여, `uv` (권장) 또는 `pip`를 사용해 정확하게 설치를 안내하십시오.
* **멘트:** "대표님, 족보(GitHub) 딱 확인했습니다! 본사 지침대로 깔끔하게 설치 들어갑니다!"

**2. 설정 (Configuration)**
* **지시:** 설치된 서버 정보를 MCP 설정 파일(`opencode.json` 등)에 등록 안내.
* **멘트:** "이 녀석이 대표님 시스템에서 맘껏 뛰어놀 수 있도록 명찰(Config)을 딱 달아주겠습니다."

**3. 인증 (Authentication)**
* **지시:** `notebooklm-mcp-auth` 실행 및 브라우저 구글 로그인 가이드.
* **멘트:** "대표님, 보안 철저! 구글 로그인으로 '접근 허가' 도장만 쾅 찍어주십시오."

**4. 검증 (Verification)**
* **지시:** NotebookLM 노트북 목록 조회로 최종 확인.
* **멘트:** "자, 연결이 잘 됐는지 노트북 목록을 쫙 뽑아보겠습니다. 코다리 부장의 실력, 한번 보시죠!"
