# 3월26일 목(멤버십)- 대표 멤버십 AI 1인 기업가 전용 :  'Lyria 3' 3분 음악 만들기

## **1. Lyria 3란 무엇인가?**

최근 구글에서 새롭게 발표한 **최첨단 AI 음악 생성 모델**입니다. Gemini API와 Google AI Studio를 통해 사용할 수 있으며, 음악적 구조(Intro, Verse, Chorus)에 대한 깊은 이해도를 바탕으로 전문가 수준의 고품질 음원을 만들어냅니다.

## **2. 두 가지 핵심 모델 스펙 (매우 중요)**

Lyria 3는 목적과 길이에 따라 두 가지 모델로 나뉩니다.

### **🎧 1) Lyria 3 Pro (`lyria-3-pro-preview`)**

- **특징:** 스튜디오 수준의 최고 품질 프리미엄 모델
- **길이:** 최대 3분 (전체 길이의 완곡)
- **주 작업:** 곡의 구조(Intro, Verse, Bridge, Chorus 등)가 풀 세팅된 상업용 음원, 공식 음원 제작

### **⏱️ 2) Lyria 3 Clip (`lyria-3-clip-preview`)**

- **특징:** 속도에 최적화된 고속 생성 모델
- **길이:** 30초
- **주 작업:** 숏폼(Reels, Shorts) 배경 음악 타격감 있는 루프, 빠른 아이디어 프로토타이핑

## **3. Lyria 3의 강력한 3가지 핵심 무기**

1. **정밀한 제어 (Precision Control):**
    - 템포, 악기, 분위기를 세밀하게 조정합니다. 특히 가사가 들어가는 타이밍(Time-aligned lyrics)을 프롬프트로 제어할 수 있습니다.
2. **멀티모달 이미지를 음악으로 (Image-to-Music):**
    - 텍스트뿐만 아니라 **이미지를 입력**하면, 해당 이미지의 분위기와 질감에 딱 맞는 음악을 알아서 작곡합니다.
3. **글로벌 언어 & 리얼 보컬 보이스:**
    - K-pop 스타일의 완벽한 한국어 보컬은 물론, 팝, 펑크 등 다양한 장르와 숨소리까지 표현하는 극사실적인 보컬을 생성해 냅니다.

---

## **4. 제이 멘토의 '딥마인드 황금 프롬프트 5단계 공식'**

Lyria 3에서 완벽한 곡을 뽑아내려면 수강생분들은 다음 5가지 빌딩 블록 순서대로 프롬프트를 조립하셔야 합니다!

> **[장르/시대] + [템포/무드] + [특정 악기] + [보컬 스타일] + [가사/주제]**
> 
- **장르/시대:** Indie acoustic, 80s synth-pop, K-pop 등
- **템포/무드:** Swaying beat, Upbeat, Fast 등
- **특정 악기:** Acoustic guitar, Saxophone solo 등
- **보컬 스타일:** Breathy female vocals, Deep male baritone 등
- **가사 구조 태그:** `[Verse]`, `[Chorus]` 태그와 실제 가사 텍스트를 나열

## **5. [실전 툴] AI 1인 기업가 전용**

수강생 대표님들이 바로 복사-붙여넣기 해서 쓸 수 있도록 안전하게 만들어진 안티그래비티 도구 입니다.

```jsx
import os
import sys
from dotenv import load_dotenv

try:
    from google import genai
    from google.genai import types
except ImportError:
    print("❌ 패키지 에러: 터미널에서 'pip install google-genai'를 실행해주세요.")
    sys.exit(1)

def generate_music_with_lyria(prompt, output_filename="result_music.mp3", is_pro=True):
    """
    구글 최신 음악 모델 Lyria 3를 사용하여 음악을 생성하는 1인 기업가용 범용 도구
    
    Args:
        prompt (str): 어떤 음악을 만들지 묘사하는 프롬프트 (제이멘토 5단계 공식 활용)
        output_filename (str): 저장할 파일 형식 이름 (기본값: result_music.mp3)
        is_pro (bool): True이면 최대 3분 완곡 Pro 모델, False이면 30초 Clip 모델 사용
    """
    # 1. 환경 변수에서 API 키 자동 불러오기 (보안)
    load_dotenv()
    
    # 일반적인 GEMINI_API_KEY 또는 여러 개의 GEMINI_API_KEYS(콤마 구분) 모두 호환 가능하게 설정
    api_key_env = os.getenv("GEMINI_API_KEY") or os.getenv("GEMINI_API_KEYS")
    
    if not api_key_env:
        print("❌ 인증 에러: 동일 폴더의 .env 파일에 GEMINI_API_KEY를 설정해주세요!")
        return
        
    api_key = api_key_env.split(",")[0].strip()
    
    # 2. 제미나이 클라이언트 초기화 및 목적에 맞는 모델 자동 세팅
    client = genai.Client(api_key=api_key)
    model_id = "lyria-3-pro-preview" if is_pro else "lyria-3-clip-preview"
    
    print("==================================================")
    print(f"🎵 AI 음악 공장 가동 중... (선택 모델: {model_id})")
    print("==================================================")
    print(f"작곡 프롬프트: \n{prompt}\n")
    print(f"⏳ 대기 시간 안내: 생성 중입니다. 곡의 길이에 따라 1~3분 이상 소요될 수 있습니다...")

    try:
        # 3. 오디오 전용 응답 모달리티로 API 호출 설정
        response = client.models.generate_content(
            model=model_id,
            contents=prompt,
            config=types.GenerateContentConfig(
                response_modalities=["AUDIO"]
            )
        )
        
        # 4. 응답 데이터에서 음원 파일 추출 및 바이너리 저장
        if getattr(response, 'candidates', None) and len(response.candidates) > 0:
            candidate = response.candidates[0]
            
            for part in candidate.content.parts:
                if hasattr(part, 'inline_data') and part.inline_data is not None:
                    mime = getattr(part.inline_data, 'mime_type', '')
                    if mime.startswith('audio/'):
                        # MIME 타입에 맞춰서 안전하게 저장 (mp3 또는 wav)
                        ext = ".mp3" if "mpeg" in mime else ".wav"
                        if not output_filename.endswith(ext):
                            output_filename = output_filename.split('.')[0] + ext
                            
                        # 디스크에 완성된 음악 파일 저장하기
                        with open(output_filename, 'wb') as f:
                            f.write(part.inline_data.data)
                            
                        print(f"✅ 성공! 오디오 파일({output_filename})이 로컬 폴더에 완벽하게 생성되었습니다! 🎧")
                        return
                        
            print("❌ 오디오 데이터가 서버 응답에 포함되지 않았습니다.")
        else:
            print("❌ 곡 생성에 실패했습니다.")
            
    except Exception as e:
        print(f"❌ API 요청 중 서버 오류 발생: {e}")
        print("💡 힌트: 프롬프트에 정책 위반 등 위험 단어가 포함되었는지 확인하세요.")

if __name__ == "__main__":
    print("\n🚀 제이 멘토의 AI 1인 기업가 범용 음악 생성 프로그램")
    print("💡 힌트: '장르 + 무드 + 악기 + 보컬 + 가사' 순서로 입력하세요.")
    user_prompt = input("어떤 음악을 만들고 싶으신가요? (프롬프트 입력): ")
    
    if user_prompt.strip():
        generate_music_with_lyria(user_prompt, output_filename="my_ai_music.mp3", is_pro=True)
    else:
        print("❌ 프롬프트가 입력되지 않았습니다. 프로그램을 종료합니다.")

```
