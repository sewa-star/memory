# WorkFlow

1. 이미지 찾아서 저작권이 있는 원본 만화 이미지 캡쳐하기 
2. 캡쳐해서 제이슨 프롬프트를 입력해서 이미지를 분석할거에요

```jsx
Analyze this webtoon/manga panel and convert all visual and narrative information into a highly detailed, structured 한글로 작성된 JSON format. Focus specifically on isolating artistic elements, character features, and composition. For each key element, extract its precise **art style** (e.g., cel-shading, painterly, line-heavy), **color palette**, and **compositional role**. Include JSON keys for:

- `art_style`: The overall rendering method and line work style.
- `atmosphere_and_mood`: The emotional tone of the scene (e.g., tense, romantic, melancholic).
- `overall_color_palette`: A list of dominant colors or hex codes used in the panel.
- `characters`: An array containing `name` (if identifiable or generic description), `expression`, `clothing_details`, and `pose`.
- `visual_elements`: An array containing `background_setting`, `lighting_source`, and `special_effects` (e.g., speed lines, sparkles, screentones).
- `composition`: Key for `camera_angle` (e.g., high angle, close-up) and `panel_layout`.

Output **ONLY** valid JSON and format the output as a copyable JSON code block using Markdown.
```

1. *무엇이 나오죠? - 분석된 이미지의 Json프롬프트가 나오겠죠! *
2. What to change? - 정확히 내가 변경하고 싶은 부분만 변경한다. 
Why? - 모두 다 변경해버리면 원본과 너무 달라지니까. 완전 새로운 이미지가 생성되니까. 
3. 그다음에 무엇을 해야하죠? 변경된 이미지를 생성을 해야하는데, 여기서 정확하게 하려면, 원본 이미지도 함께 추가해주면 좋아요. 
4. 여기서 나온 이미지를 가지고 영상생성을 하면 됩니다.
