## 🏛️ 1. 차세대 AI 패러다임 (Paradigm)

| **키워드** | **정의 및 컨퍼런스 맥락** |
| --- | --- |
| **에이전트형 AI**
(Agentic AI) | 질문에 답만 하는 수준을 넘어, 목표를 달성하기 위해 스스로 계획을 세우고 도구를 사용해 실행하는 **'능동적 AI'** 체계. |
| **바이브 코딩**
(Vibe Coding) | 복잡한 프로그래밍 언어나 문법 대신, 자연어로 내가 원하는 결과물의 '느낌(Vibe)'을 설명하여 앱이나 기능을 즉시 구현하는 개발 방식. |
| **물리적 AI**
(Physical AI) | 가상 세계를 넘어 로봇 등 물리적 실체에 탑재되어 현실 세계를 이해하고 이동하며 작업을 수행하는 **'체화된(Embodied) AI'**. |
| **온디바이스 AI**
(On-device AI) | 서버(클라우드)를 통하지 않고 기기(모바일, 웹, 데스크톱) 내부에서 직접 AI 모델을 구동하여 **보안성, 저지연, 오프라인** 성능을 확보하는 기술. |

## 🚀 3. 주요 모델 및 API (Models)

| **키워드** | **정의 및 컨퍼런스 맥락** |
| --- | --- |
| **Gemma (젬마)** | 구글의 강력한 기술력을 바탕으로 만든 **오픈 모델 제품군**. 개발자가 자유롭게 수정하고 배포 가능. |

| **Google AI Edge** | 모바일, 웹 등 기기 끝단(Edge)에서 AI를 실행하기 위한 구글의 통합 제품군. |

## 🎨 4. 생성 미디어 (GenMedia Suite)

| **키워드** | **모델 역할** |
| --- | --- |
| **Veo** | 텍스트나 이미지를 기반으로 고품질 영상을 생성하는 **비디오 모델**. |
| **Nano Banana** | 이미지 생성 및 편집을 담당하는 최신 이미지 모델 (**Gemini 3 Flash Image**). |
| **Lyria** | 전문가 수준의 편곡과 보컬 성능을 지원하는 **음악/오디오 생성 모델**. |
| **Genie** | 텍스트나 이미지를 기반으로 인터랙티브한 환경(게임 등)을 생성하는 모델. |
