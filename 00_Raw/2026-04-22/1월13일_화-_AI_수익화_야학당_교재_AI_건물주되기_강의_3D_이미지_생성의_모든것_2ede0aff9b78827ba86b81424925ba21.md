## 1. [기초] 스타일라이즈드 3D 캐릭터 공식 (Rubberhose Style)

빈티지하면서도 현대적인 느낌을 주는 'Rubberhose(고무 호스)' 스타일의 3D 캐릭터를 생성하는 마스터 프롬프트입니다. + 이미지를 함께 첨부하면 이미지에 맞는 캐릭터가 생성됩니다.

> 💡 프롬프트 공식:
> 

```jsx
[주제], stylized 3D rubberhose character, smooth 3D render, Solid [색깔] color background, dynamic pose, soft ground shadow, bright soft lighting, modern playful design
```

- **Rubberhose Character:** 1930년대 애니메이션 스타일(미키 마우스 등)을 3D로 재해석하여 관절이 없는 듯 부드럽게 휘어지는 팔다리를 표현합니다.
- **Solid Background:** 피사체에 집중할 수 있도록 배경을 단색 처리합니다.
- **Modern Playful Design:** 고전적인 스타일에 현대적인 깔끔함을 더해줍니다.

## 2. [심화] 복합 요소 제어: 파라미터 프롬프팅 (JSON)

단순한 문장보다 더 정교하게 의상을 입히고, 특정 텍스트를 포함하고, 상황을 연출하기 위해 구조화된 데이터를 사용합니다.

**🎯 목표:** 사모예드 캐릭터가 특정 브랜드 모자를 쓰고 텍스트를 출력하는 복합 장면 생성

- **가독성(Visibility):** 캔의 라벨, 모자의 자수, 말풍선의 텍스트가 뭉개지지 않고 선명하게 나오도록 `clearly visible`과 `legible` 키워드를 강조합니다.
- **구조화:** 복잡한 요구사항을 JSON 형태로 입력하면 AI가 각 요소를 더 정확하게 인식하는 경향이 있습니다.

```jsx
{
  "camera_settings": {
    "style": "cinematic",
    "lens": "50mm standard lens", 
    "shot_type": "close-up"
  },
  "subject": {
    "type": "Samoyed dog",
    "attributes": [
      "smiling and happy expression"
    ],
    "clothing": "cute hat with 'Connect AI LAB' embroidered on it",
    "action": "holding and showcasing a can close to the camera with its paw",
    "speech_bubble": {
      "content": "AI 멘토 제이 최고!",
      "style": "cute and friendly comic book style speech bubble"
    }
  },
  "environment": {
    "location": "beach",
    "background": "ocean",
    "weather": "beautiful sunny day"
  },
  "requirements": {
    "visibility": "All writing on the can, hat, and speech bubble is clearly visible and legible"
  }
}
```
